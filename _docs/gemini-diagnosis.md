# 卡冈图亚黑洞 — Gemini 视觉诊断

## 核心结论

**4 万离散粒子永远达不到《星际穿越》的 Gargantua 效果。** DNEG 团队用的是自定义体积光线追踪器（DNGR），计算数百万条光线穿过连续发光等离子体。离散粒子方案的本质局限是：

1. 粒子间有空隙 → 眼睛能分辨单个点 → 像尘埃云不是等离子盘
2. 屏幕空间后处理无法做真正的引力透镜 → 被遮挡的盘后光线已丢失
3. 缺乏 HDR 辉光 → SDR 画面，事件视界硬边界

## 三大视觉缺陷

| 问题 | 原因 |
|---|---|
| 缺失上下透镜光晕 | 黑色球体遮挡了后面的粒子，屏幕空间透镜无法恢复已丢失的像素 |
| 粒子稀疏像尘埃 | 25K 粒子在 2.0-16.0Rs 范围 → 密度极低 |
| 无 HDR Bloom | 内盘不够亮，没有光溢出到视界上 |

## 如果坚持粒子路线

- **粒子数**：25 万–50 万（最低）
- **材质**：`PointsMaterial` + 模糊径向渐变精灵贴图 + `AdditiveBlending` + `depthWrite:false`
- **厚度**：`y = (random-0.5) * thickness * r`（高斯分布，外厚内薄）
- **透镜 trick**：在**顶点着色器**中弯曲 `gl_Position`（不是屏幕空间后处理），让粒子在光栅化前被扭曲
- **HDR**：内盘颜色值超过 1.0 + UnrealBloomPass

## 推荐替代方案：Raymarching

全屏 quad + 单个 GLSL 片段着色器：

1. 从相机发射光线
2. 用 Schwarzschild 度量弯曲光线
3. 沿弯曲路径采样 3D 噪声纹理（吸积盘密度）
4. 累积颜色和发光（体积渲染）

这是 Singularity (MisterPrada) 的方案，也是浏览器端唯一能达到电影级效果的方法。

**优点**：真正的引力透镜、连续体积等离子、不需要后处理 hack  
**代价**：完全不同的技术路线，需要重写整个渲染管线
